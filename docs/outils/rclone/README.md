# rclone

[rclone](https://rclone.org/) est un outil en ligne de commande permettant de gérer les fichiers sur différents services de stockage cloud.

## Installation

* [rclone.org - Install](https://rclone.org/install/)
* [rclone/install.sh](https://github.com/mborne/mborne.github.io/blob/main/docs/outils/rclone/install.sh) assure le téléchargement et l'installation du .deb :

```bash
curl -sS  https://mborne.github.io/outils/rclone/install.sh
```
