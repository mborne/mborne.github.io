# restic

[restic](https://restic.net/) est un utilitaire de sauvegarde avec une approche incrémentale.

## Installation

[restic/install.sh](https://github.com/mborne/mborne.github.io/blob/main/docs/outils/restic/install.sh) assure le téléchargement du binaire en `/usr/local/bin/restic` :

```bash
curl -sS https://mborne.github.io/outils/restic/install.sh | bash
```

## Utilisation

Voir [restic.net - Quickstart](https://restic.net/#quickstart)
