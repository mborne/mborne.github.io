# DBeaver

[DBeaver Community](https://dbeaver.io/) est un client OpenSource supportant de nombreuses bases de données (sqlite, DuckDB, PostgreSQL, MySQL, OpenSearch...)

## Installation

* [dbeaver.io - Download](https://dbeaver.io/download/)
* [dbeaver/install.sh](https://github.com/mborne/mborne.github.io/blob/main/docs/outils/dbeaver/install.sh) assure l'installation à l'aide d'un package :

```bash
curl -sSL https://mborne.github.io/outils/dbeaver/install.sh | bash
```
