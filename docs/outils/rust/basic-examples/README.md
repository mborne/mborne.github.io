# Rust - Quelques exemples rédigés pour découvrir le langage

## Usage

```bash
rustc print-vars.rs
./print-vars "john doe" 45
```

* [helloworld.rs](helloworld.rs) : Affichage de "Hello world!" dans la console.

* [arrays.rs](arrays.rs) : Manipulation de tableaux

## Basic IO
