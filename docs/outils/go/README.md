# Go

!!!info "Motivation"
    Le langage Go est de plus en plus utilisé dans le monde "DevOps" (docker, terraform,...) et il commence à être utilisé en géomatique (ex : [pg_featureserv](https://github.com/CrunchyData/pg_featureserv)).
    Il est donc tentant de prendre un peu de temps pour l'apprendre sérieusement...

## Installation

* [go/install.sh](https://github.com/mborne/mborne.github.io/tree/main/docs/outils/go/install.sh) assure l'installation à partir des sources :

```bash
curl -sS https://mborne.github.io/outils/go/install.sh | bash
```
