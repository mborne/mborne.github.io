    Données spatiales¶

    • OWSLib pour accès aux services OGC (WFS, WMS,...)
      • examples/owslib/wfs-gpf-filtering.py
    • Shapely pour des calculs géométriques (basée sur GEOS).
    • GeoPandas aggrégation de plusieurs bibliothèques.
    • Rasterio pour les GeoTIFF.
    • GDAL est un binding de la bibliothèque GDAL/OGR (version "roots" des autres bibliothèques)
    • pyproj offre des fonctionnalités de transformations de coordonnées (binding PROJ4 pour Python).

    Rendu cartographique¶

    • Folium assure le rendu des données cartographiques en générant une carte Leaflet.
    • Descartes assure le rendu des géométries à l'aide de matplotlib.
