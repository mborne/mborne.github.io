### Configurer nano pour indenter facilement

Pour `kubectl edit` avec `EDITOR=nano`, définir `.nanorc` comme suit permet d'utiliser TAB et SHIFT+TAB sur plusieurs lignes :

```bash
set tabstospaces true
set tabsize 2
```
